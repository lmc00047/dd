Knowledge Induction from Experimental Data

Generate a Knowledge Base from Experimental Data

This step creates a knowledge base from experimental data.
As a simple validation strategy we randomly select 80% of data samples for training and the remaining 20% for test. So for, GUAJE software allows us to automatically separate the dataset into training and test datasets.
Then, all the variables and rules will be defined over it.
SOFTWARE VIEW (STEP BY STEP):
1. Menu KB -> New
2. Menu Data -> Open
3. Select the desirable dataset (in our case DATA/IRIS.txt)
4. Select all the variables (press NO)
5. Select 1 variable as output
6. Menu Data -> Generate -> Sample -> 1/80%
7. Menu KB -> Close
8. Menu KB -> Open
9. Select the desirable file (in our case DATA/DATA0/IRIS.txt0.ikb.xml)
