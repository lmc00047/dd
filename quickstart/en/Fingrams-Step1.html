Creation of Fingrams from a Knowledge Base

Initialize the Knowledge Base from Data

This step creates a knowledge base from experimental data. Then, all the variables and rules will be defined over it.
SOFTWARE VIEW (STEP BY STEP):
1. Menu KB -> New
2. Menu Data -> Open
3. Select the desirable dataset (in our case FINGRAMS/WINE.txt)
4. Select all the variables (press NO)
5. Select 1 variable as output
