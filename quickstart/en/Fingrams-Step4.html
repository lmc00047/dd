Creation of Fingrams from a Knowledge Base

Generate Rules

This step allows to induce a set of rules from the loaded training data.
The software allows the use of some methods to automatically generate rules from data. Here, Fuzzy Decision Trees method with prunning is used to generate a set of rules. Default parameters are considered (Maximum tree depth = 5).
In case you click "DO", then please wait until the expert window is updated with the generated rules before going to next step.
SOFTWARE VIEW (STEP BY STEP):
1. Button Data -> Rules -> Fuzzy Decision Trees
2. Press "Yes".
3. Press "Default values" and "Apply"
4. Save the pruned set of rules (39 rules if everything was OK)
