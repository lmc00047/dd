Design of a Fuzzy KB Supported by Fingrams

Partition Optimization

This step allows to optimize the accuracy of the system. GUAJE provides two methods to optimize accuracy through partition tuning:
  • Solis-Wets
  • Genetic Tuning
For instance, if you run "Solis-Wets" the fuzzy partitions are slightly tuned looking for an accuracy improvement.
If you click "DO", then please be patient. This process may take long. Once optimization is carried out, the fingrams are updated. Wait until the fingram windows are automatically re-displayed.
STEPS SOFTWARE:
1. Button Expert (if needed)
2. Button Optimization -> Partition Tuning -> Solis Wets -> Automatic
3. Yes -> Label by Label / No bounded limits / 100 iterations
4. START
