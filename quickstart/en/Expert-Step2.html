Expert Knowledge Extraction and Representation

Create a new Input Variable

All the input variables must be created in this step.
The new inputs can be named and defined by different types (numerical, logical and categorical). For each input different number of labels and linguistic terms are allowed.
Moreover, it can be selected the modal points in each case, changing the default equidistant ones.
In case you click "DO", then please wait until the inputs definition window is open before going to next step.
SOFTWARE VIEW (STEP BY STEP):
1. Open the Expert View (if needed)
2. Menu Inputs -> New input
3. Type -> Select the desirable type
4. Labels -> Number of labels
5. Close
6. Repeat from point 2 until you obtain the number of inputs needed
