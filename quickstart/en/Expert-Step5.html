Define Rules

This step allows to define the rules that will control the behavior of the system.
Each rule will cover part of the input space. Several rules can be defined, but the larger the number of rules, the larger the complexity of the system.
SOFTWARE VIEW (STEP BY STEP):
1. Open the Expert View (if needed)
2. Menu Rules -> New rule
3. Rules -> Select the desirable antecedents and consequents for the rule
4. Repeat from point 2 until you obtain the number of rules needed
