Design of a Fuzzy KB Supported by Fingrams

Generate Fuzzy Partitions

In this step, fuzzy partitions are automatically generated from data.
Partitions are generated with the so-called Hierarchical Fuzzy Partitioning (HFP) algorithm. Five labels are defined for each input variable. If needed, you can change the generated partitions.
If you click "DO", then please be patient. This process may take long. Wait until the expert window is automatically open.
STEPS SOFTWARE:
1. Button Data -> Partitions -> All inputs (Default Parameters)
2. Select "Type of partition" equals "HFP" and "Number of labels" equals 5, in "Select partition" section
3. Once the desirable configuration is set, press "Apply" button to updated the selected partition
4. Repeat from step 2 until all the variables have been treated
