Design of a Fuzzy KB Supported by Fingrams

Rule Base Simplification

This step allows to simplify the system previously generated. GUAJE provides three methods to simplify a fuzzy rule-based system:
  • Linguistic Simplification
  • Genetic Rule Selection
  • Logical View Reduction
For instance, if you first run "Logical View Reduction", then the number of rules drops from 37 (assuming you deactivated rules R1, R23, R35 and R37 in the previous step) to 10. Afterwards, you can run "Linguistic Simplification". As result, the number of active rules is reduced up to 6 while the number of inputs passes from 13 to 4. As it can be appreciated below in the "Quality Evaluation" panel section, all quality indicators are improved (or at least kept they remain the same before and after simplification).
If you click "DO", then please be patient. This process may take long. Once simplification is carried out, the fingrams are updated. Wait until the fingram windows are automatically re-displayed.
STEPS SOFTWARE:
1. Open the Expert View (if needed)
2. Button Logical View
3. Button Simplification -> Automatic
4. No -> Yes -> Default Values -> Apply
5. START
6. OK, OK, OK
