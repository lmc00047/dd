Set Fuzzy Operators

This step allows to define the fuzzy operators used for t-norms, t-conorms and the defuzzyfication method.
This operators will be determined by the output type (numerical/categorical).
It can be selected:
  • For the conjunction: Minimum, product, and Lukasiewicz.
  • For the disjunction: Maximum and sum.
  • For the defuzzyfication method: Area, mean max, Sugeno and max crisp.
In this illustrative example we consider the "Default values" (minimum, maximum, max crisp). Anyway, you are free to try with the alternative operators.
SOFTWARE VIEW (STEP BY STEP):
1. Button SET FIS OPERATORS
2. Operator of conjunction (t-norm)
3. Operator of disjunction (t-conorm)
4. Method of defuzzification
