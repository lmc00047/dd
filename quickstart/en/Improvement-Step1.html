Design of a Fuzzy KB Supported by Fingrams

Generate a Knowledge Base from Experimental Data

This step creates a knowledge base from experimental data, splitting the available data into training and test datasets.
As a simple validation strategy we will divide the dataset (75% training, 25% test).
Then, all the variables and rules will be defined over it.
STEPS SOFTWARE:
1. Menu KB -> New
2. Menu Data -> Open
3. Select the desirable dataset (in our case IMPROVEMENT/WINE.txt)
4. Select all the variables (press NO)
5. Select 1 variable as output
6. Menu Data -> Generate -> Sample -> Number of samples: 1 / Validation/data ratio: 75% / Seed: 1
7. Menu KB -> Close
8. Menu KB -> Open
9. Select the desirable file (in our case IMPROVEMENT/IMPROVEMENT0/WINE.txt0.ikb.xml)
