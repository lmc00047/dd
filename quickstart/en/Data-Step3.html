Knowledge Induction from Experimental Data

Generate Partitions

In this design step, fuzzy partitions are automatically generated from data.
Partitions are fitted to available data for each variable. Three labels are considered for each fuzzy partition. Of course, if needed, you are free to change the generated partitions.
In case you click "DO", then please wait until the expert window is open before going to next step.
SOFTWARE VIEW (STEP BY STEP):
1. Button Data -> Partitions -> All inputs
2. Select the adequate partition in "Select partition" section
3. Once the desirable configuration is set, press Apply button to updated the selected partition
4. Repeat from step 2 until all the variables have been treated
