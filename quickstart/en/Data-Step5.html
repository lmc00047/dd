Knowledge Induction from Experimental Data

Generate Rules

This step allows to induce a set of rules from the loaded training data.
There are some methods to automatically generate rules from data. Here, Wang and Mendel method is used to generate a set of rules.
In case you click "DO", then please wait until the expert window is updated with the generated rules before going to next step.
SOFTWARE VIEW (STEP BY STEP):
1. Button Data -> Rules -> Wang and Mendel
2. Save the set of rules generated
