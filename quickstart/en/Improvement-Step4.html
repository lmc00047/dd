Design of a Fuzzy KB Supported by Fingrams

Generate Fuzzy Rules

This step allows to induce a set of rules from the loaded training data.
The software allows the use of some methods to automatically generate rules from data. Here, Fuzzy Decision Trees method with prunning is used to generate a set of rules.
STEPS SOFTWARE:
1. Button Data -> Rules -> Fuzzy Decision Trees
2. Press "Yes".
3. Press "Default values" except for MaxTreeDepth=3, and "Apply"
4. Save the pruned set of rules (41 rules if everything was OK)
